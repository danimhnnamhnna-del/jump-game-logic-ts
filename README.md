# jump-game-logic-ts
A TypeScript project to solve the Jump Game problem. It determines if you can reach the last index of an array based on maximum jump lengths.
# Jump Game Solution

A simple and efficient TypeScript implementation to solve the **Jump Game** challenge using a Greedy Algorithm.

## Description
This program determines if it's possible to reach the last index of an array, where each element represents the maximum number of steps you can jump forward from that position.

## Logic
- **Algorithm:** Greedy Approach.
- **Time Complexity:** O(n).
- **Space Complexity:** O(1).
